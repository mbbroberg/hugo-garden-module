## [1.0.0] - 2025-11-01

### Added
- Initial release of Hugo Garden Module
- Garden list layout with browsing interface
- Browse by growth stage (seed, sprout, rhizome)
- Browse by topic/tags
- State indicator partial for single posts
- Garden callout partial for linking back to garden
- Comprehensive CSS with CSS variable support
- Dataview-style metadata JavaScript styler
- Garden archetype for creating new entries
- Full configuration system via hugo.toml
- Customizable colors, emojis, and labels per growth stage
- Responsive design for mobile and desktop
- Animated entry cards
- Complete example site with three sample posts
- Comprehensive README documentation
- Contributing guidelines
- MIT License

### Features
- Three default growth stages with customization support
- Tag-based organization and browsing
- Date tracking (planted and last tended)
- Visual growth indicators
- Smooth animations and transitions
- Dark mode support via CSS variables
- Grid layout with auto-fill responsive columns
- Staggered entry animations
- Badge-based navigation
- Configurable callout messages
